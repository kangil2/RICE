## **구현 파트**

- 이강민 (상세페이지 및 리스트페이지)
  ```
  - 상품 리스트 페이지
  - 상품 리스트 정보 API
  - 상품 상세 정보 페이지
  - 상품 상세 정보 API
  - error 모달창 및 버튼 상호작용 기능
  ```

- 박준양 (상세페이지 및 리스트페이지)
  ```
  - 메인 페이지
  - 검색 바
  - 상품 리스트 필터링
  - 카테고리 메뉴
  ```

- 이찬호 (상세페이지 및 리스트페이지)
  ```
  - 장바구니 페이지
  - 장바구니 추가/삭제/수정/조회 API
  ```

- 최지혁 (상세페이지 및 리스트페이지)
  ```
  - 회원 정보 페이지
  - 주문 페이지
  - 결제 OPEN API
  ```

- 황현우 (상세페이지 및 리스트페이지)
  ```
  - 로그인 페이지
  - 소셜 로그인 OPEN API
  ```
